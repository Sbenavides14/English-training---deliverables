\# Purchase Calculator in Python

\## Description

This project is a simple Python program that calculates the total cost
of a purchase.

It is a basic example designed to demonstrate how user input, variables,
and arithmetic operations work in Python.

The program receives information about a customer and a product, then
calculates the subtotal, discount, and final total.

It uses simple mathematical operations and prints the results on the
screen.

Currently, the program is processing user input to calculate the
purchase values in real time.

\## How it works

1\. The program asks the user to enter the customer name.

2\. The program requests the product name.

3\. The system asks for the product price and converts the value to a
decimal number using \`float\`.

4\. The program asks for the quantity of the product and converts the
value to an integer using \`int\`.

5\. The system calculates the subtotal by multiplying the product price
by the quantity.

6\. The program calculates a discount based on 10% of the purchase.

7\. The system calculates the final total by subtracting the discount
from the subtotal.

8\. Finally, the program prints the subtotal, discount, and total
purchase value.
